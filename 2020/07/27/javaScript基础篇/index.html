JavaScript 实现方式（各浏览器引擎）

ECMAScript 是JavaScript的标准，但这俩基本是一个意思 Google Chrome （V8引擎） 目前执行js最快的引擎

JS的特点：

-解释型语言
-类似于C和Java的语法结构
-动态语音
-基于原型的面向对象

JavaScript 基本语法

        alert('弹出一个内容框')
        document.write('可以向body里输出一个内容')
        console.log('向控制台输出一个内容')

        1.JS中严格区分大小写
        2.JS中每一条语句已分号(;)结尾 -如何不加分号，会消耗系统资源，有一些时候会加错分号
        3.JS会忽略多个空格和换行，所以可以利用空格和换行来进行代码格式化
   字面量,都是一些不可改变的值  都是可以直接使用的
   变量,可以用来保存字面量，而且变量可以任意变量

标识符

   -在JS中可以由自己随意命名的称为标识符
   -命名规则:
     1.标识符中可以含有数字、字母、_、$
     2.标识符不能以数字开头
     3.标识符不能是ES中的关键字或保留字 例:(var var = 123)
     4，规范  -首字母小写,每个单词开头字母大写,其余字母小写

数据类型

   在JS中一共有六种基本数据类型
   String 字符串
   Number 数值
   Boolean 布尔型
   Null 空值
   Undefined 未定义
   Symbol 原始数据类型

   引用数据类型
   Object 对象

   如何使用？
   -使用定义字符串 (在JS中使用引号引起来)
   const str = 'hello'

转义字符

   \' 表示 '
   \" 表示 "
   \n 表示 换行
   \t 表示 制表符 相当于按下Tab
   \\ 表示 \

   typeof  检查一个变量的类型

   JS中表示最大的值
   Number.MAX_VALUE

   布尔值
   布尔值只有两个,主要是做逻辑判断
   true 真
   false 假

   Null 类型 只有一个 null
   -null这个值，专门用来表示一个为空的对象

   Undefined(未定义) 
   -当声明一个变量，但是并不给变量赋值时，他得知就是undefined

强制转换类型

方式:
      - 调用被转换数据类型的toString()方法     调用方法 就是 xxx.yyy方法()
      - 调用String() 函数 转换         函数 使用  String(参数)

强制转数字
方式:  
      -调用Number()
      注意: 
           1.如果纯数字的字符串,则直接转换称 --> 数字
           2.如果字符串中有其他字符,则转换成 --> NaN
           3.如果字符串为空格或空字符串,转换成 --> 0
           4.如果变量为 null,转换成 --> 0
           5.如果变量为 布尔值 则  true --> 1, false --> 0
           6.如果变量为 undefined 则转换成 NaN


方式二: 
      -这种专门用来对付字符串  (非字符型  会先转换成字符型再进行转)
      -parseInt()  转换成  number  
      -parseFloat()  转换成 小数型


      Boolean()
      数字转布尔   除了 0 和 NaN  都是true
      字符串转布尔  除了 ''  都是true
      null,undefined 转 为 false

运算符

        进行运算符操作时   首先会先把 变量转换成数字 再进行 运算
        任何值和NaN做运算，都是NaN

        如果对两个值 做加法运算，有非字符串 ，做拼接操作  例(    let a = 100 +'1' -'100')  //901
        隐式转换类型
        如果是减法运算， 便是做转换数值 (Number类型) 减法
逻辑运算符
         &&   如果第一个为true    返回第二个     let result= 2 && 0       返回false那个 0
         &&   如果第一个为false   返回第一个     let result= 0 && 2       返回false那个 0
         ||   如果第一个为true    返回第一个     let result= 1 || 0       返回true 那个 1
         ||   如歌第二个为false   返回第二个     let result= 0 || 1       返回true 那个 1

相等运算
         NaN不与任何相等
         undefined == null  //true
         isNaN()  检查一个数是不是NaN

if,switch相关语法

语法: if() {

}
-if语句只能控制紧跟其后的那一条语句
-if语句想要控制多条语句，可以将语句写进 代码块中{ }
-if语句后代码块不是必须的，但为了条理清楚，尽量加上代码块格式

语法二: if(条件表达式){
    语句
} else if(条件表达式){
    语句
} else {

}

-从上往下一次进行判断
-如果是true,执行当前代码块语句
-如果是false,依次向下判断下一个if语句
-如果都不满足执行 else 语句块

语法: switch(值) {
    case 1: 
    表达式
    break;
    case 2:
    表达式
    break;
    case 3:
    表达式
    break;
    default:
    表达式
    break;
}
-使用break来退出switch
-default  所有的都无法成功时,调用default

弹出框类型

 prompt('请输入小明的期末成绩:');
  页面弹出一个带有输入框的弹框。
 用户输入的内容作为函数的返回值返回，可以直接定义一个变量来接收该内容
 alert();  页面弹出一个弹框

循环语句

循环语句: 通过循环语句可以反复的执行一段代码多次

while循环
 -语法: 
       while(条件表达式) {
           语句...
       }

       break 可以退出循环



for循环
1.初始化表达式
2.条件表达式
3.更新表达式

for循环语法: 
for(①初始化表达式;②条件表达式;④更新表达式) {
    ③语句...
}
   for循环执行流程:
   ①执行初始化表达式，初始化变量
   ②执行条件表达式，判断是否执行循环。
     如果为true，则执行循环③
     如果为false，则终止循环
   ④执行更新表达式，更新表达式执行完毕继续重复②



  for循环中的三个部分都可以省略 也可以写在外部  
  死循环:
  for(;;){
      alert('hello')
  }
